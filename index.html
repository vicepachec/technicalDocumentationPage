Css Selectors

We use this selectors to help us with designing the page with CSS, so that we can be more specific about which elements we want to change.

Here are some examples:

  • Choosing by element: we can select an element to change every instance of that element at once by using the name of the element we want to change, like for h1 we would simply write h1{} and then we put the styles within the brackets.
  • Change everything: we can use the * selector to select and change every single element within the hmtl code.
  • Choosing by id: after we define and id for an element for example <ph id="paragraph"> we can change that single element by selecting it's id to choose an ID we use the # symbol like #paragraph{} and then we can style only the elements that have that id.
  • Choosing by class: this follows the same logic of the id but with classes for example using <div class="container"> we then can select only the elements with that class, by using .container{} and choosing the styles we want.
  • Specificity: something important to know is that certain selectors are more specific than others, if we have an <div class="container"> and we use div{background-color:white} and .container{background-color: yellow} how will the code know which one to use? That's when specificity comes into play, the order goes like this id>class>element, so in the example i gave the background-color would be yellow, since classes are more specific than elements.

Display Property

Display Property define how we want that element to behave since all of them have a default way of displaying with this we can change it, so we can style things the way we prefer

For display we have a few options to use.

  • Inline: make the element behave as and Inline object, making it ignore width, height, margin and padding, only pushes elements horizontally, for this we use display: inline;
  • Block: make the element take an entire block of the document, also respects width, height, margin and padding, for this we use display: block;
  • Inline-block: make the element and inline object, but have it respect width, height, margin and padding, for this we use display: inline-block;

Borders

We can also define borders to our elements to create separation or style our elements.

  • Border-widht: we use this to define the widht of the border like for example border-width: 2px;
  • Border-style: we use this to define the style of the border like for example border-style: dotted;
  • Border-color: we use this to define the color of the border like for example border-color: red;
  • Border: we use this one to select all three at once for example border: 2px dotted red;
